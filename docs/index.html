产品能力（扁平、可追溯）

围绕训练链路的每个关键节点：数据、训练、评估、流水线与报告。

1

数据集清洗与准备

本地上传或从 URL 导入，清洗后生成就绪数据集并进入训练候选。

2

训练任务管理（经典版）

可查看进度、日志、重启/删除等操作；训练完成后产出模型。

3

评估与报告闭环

评估指标与报告（预览/下载）贯穿任务生命周期，支持错误原因洞察。

4

Agent 版智能画布

分步配置：先确认目标，再准备数据，再启动流水线编排。

5

一键流水线（2.0 主线）

Coordinator 异步执行清洗 → 训练 → 评估，并持久化流水线实例状态。

6

双版本互通

Agent 与经典版共享同一套数据表；切换不丢结果，易于演示与复用。
