# 截图占位目录（用于 GitHub Pages）

GitHub Pages 页面：`docs/index.html`

将你的产品截图复制到本目录，并按下面文件名替换：

- `01-workflow.png`：工作台/最近动态
- `02-datasets.png`：数据集管理/清洗结果
- `03-training-jobs.png`：训练任务管理
- `04-evaluation.png`：模型评估管理与报告
- `05-agent-canvas.png`：Agent 画布步骤配置
- `06-data-agent-plan.png`：Data Agent 规划选项

建议：截图保持宽度在 1200px 左右，压缩为 PNG 或 WebP（体积越小页面越快加载）。
